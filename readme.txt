Easily add Creative Commons licensed images to your wordpress posts.

== Description ==

This plugin allows your users to easily download Flickr photos that are licensed under Creative Commons
to the local server for attaching to posts.

Features include:
  * Customizable Attribution Text
  * Custom CSS class on Attribution & Image
  * Downloads the image to your server
  * Custom naming of image filename and alt text
  * Two ways to search - from within the sidebar and from Add Media

 NOTE: This plugin requires PHP5.

== Installation ==

 1. Upload the folder iflickr to the `/wp-content/plugins/` directory
 2. Activate the plugin through the 'Plugins' menu in WordPress
 3. Set the options for the plugin, including the Flickr API key, in the 'Options' menu in WordPress
